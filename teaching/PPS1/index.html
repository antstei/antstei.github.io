Strings in C …

  • sind Arrays vom Datentyp char
  • werden mit einem \0 – dem Nullzeichen – terminiert

Strings in C …

  • benötigen ein char-Array mit (mindestens) Zeichenanzahl + 1 Elementen
  • können mit einem String-Literal (" … ") initialisiert werden

Aufgabe 1

Erstelle ein Programm, das die Länge eines beliebigen C-Strings ohne Nullzeichen ausgibt.

Strings in C …

  • können mit der Standardfunktion fgets() eingelesen werden
  • können mit der Standardfunktion fputs() ausgegeben werden

Aufgabe 2

Erstelle ein Programm, das die Position eines bestimmten Zeichens in einem beliebigen C-String ausgibt. Wir gehen davon aus, dass das zu suchende Zeichen im C-String enthalten ist.

Aufgabe 2½

Erstelle ein Programm, das die Position eines bestimmten Zeichens in einem beliebigen C-String ausgibt, sofern dieses im String enthalten ist.

(☕-)Pause

Strings in C …

  • können nicht mit einem Zuweisungsoperator (=) kopiert werden

Aufgabe 3

Erstelle ein Programm, das einen beliebigen C-String in ein anderes char-Array kopiert und dieses anschließend ausgibt.

Strings in C …

  • können mit Funktionen der Standard-Headerdatei string.h (unter anderem) kopiert, zusammengefügt und verglichen werden:
char* stcpy(s, cs) C-String cs inklusive \0 in Array s kopieren; liefert s.
char* strcat(s, cs) C-String cs hinten an die C-String s anhängen; liefert s.
char* strlen(cs) Liefert die Länge exklusive \0 vom C-String cs.
